# Tools & calculators

Beyond search and analysis, MatFinder bundles several everyday utilities.

## CIF editor

View and edit Crystallographic Information Files, and generate symmetrized or
standardized variants ready to feed into the analysis module.

## Stoichiometric calculator

Balance chemical equations, identify the limiting reagent, and compute theoretical
and percent yields, plus solution molarity. Useful for planning a synthesis whose
product you will later characterize.

## Mass-proportion calculator

Given a target compound and total sample mass, compute the individual element/reagent
masses to weigh out.

## Interactive periodic table

Pick elements visually for a search or a calculation, with categories, atomic numbers
and atomic masses at a glance.

## Open-access article retrieval

Fetch the open-access PDF for a given **DOI** via **OpenAlex** and **Unpaywall**, and
run quick literature lookups — handy when you find a candidate structure and want the
paper behind it.
